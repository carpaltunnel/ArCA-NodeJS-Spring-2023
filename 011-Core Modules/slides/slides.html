Core Modules





Let's start exploring the core modules that are bundled with Node.js


NOTE: These examples are no longer executable in your browser javascript console since it lacks the Node.js engine and therefore the core modules.

require('module-name')

To be able to use most modules, you need to reference them in your script. This is done with the require() function.

const filesystem = require('fs');
// Now you can use the Filesystem module referenced by variable name "filesystem"

fs - Filesystem

The fs or "File System" module provides an interface to the computers file system. As usual, we cover only a small subset of functionality here. And, we're starting with only synchronous functions. *nix based naming

  1. fs.existsSync(file) - check if file exists
  2. fs.writeFileSync(file, data) - write "data" to file
  3. fs.readFileSync(file) - read file data into variable
  4. fs.readdirSync(dir) - read contents of dir into an array
  5. fs.renameSync(file, newfile) - rename file to newfile
  6. fs.rmSync(file) - 'rm' (delete) file

process

The process module gives you access to the current process. Process is a global object - no require() needed. *nix based naming.

  1. .cwd() - Current working directory
  2. .chdir(dir) - Change to directory dir
  3. .env - Environment variables
  4. .pid - Current process ID
  5. .platform - Current platform
  6. .exit(code) - Exit process with specified return code

os - Operating System

The os module provides information about the operating system.

  1. os.arch() - Current OS architecture
  2. os.platform() - Current platform
  3. os.totalmem() - Total memory available (bytes)
  4. os.freemem() - Free memory available (bytes)
  5. os.homedir() - Current user's home directory
  6. os.cpus() - List of CPUs and properties of them

ZLib



ZLib is a core module that provides compression and decompression functionality. It only supports GZip, Deflate, and Brotli protocols. So, while there is an unzip() function, this does not support the "zip" compression algorithm.

ZLib Functions

  1. zlib.gzipSync() - sync compression using GZip
  2. zlib.gunzipSync() - sync decompression using GZip
  3. zlib.brotliCompressSync() - sync Brotli compression
  4. zlib.brotliDecompressSync() - sync Brotli decompression
  5. zlib.deflateSync() - sync Deflate compression
  6. zlib.inflateSync() - sync Deflate (Inflate) decompression
